
### Installation

Clone repo:

```sh
git clone https://github.com/chipcop106/ElearningNextJs.git
cd ElearningNextJs
```

Install the dependencies:

```sh
yarn install
```

or

```sh
npm install
```

### Development Workflow

Start a live-reload development server:

```sh
yarn dev
```

or

```sh
npm run dev
```

Generate a production build:

```sh
yarn build
```

or

```sh
npm run build
```

Generate a static file:

```sh
yarn export
```

or

```shell script
npm run export
```
